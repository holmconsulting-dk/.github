## Repositories

  ### [claude-multiregnskab](https://github.com/holmconsulting/claude-multiregnskab)                                                      
  A Claude Code plugin that brings Multiregnskab into your AI-assisted workflow. Instead of memorizing CLI flags, you describe what you need
   in plain language — in Danish or English — and Claude handles the rest, using `cli-multiregnskab` under the hood. Install it once and    
  your Claude Code sessions become a natural-language interface to your accounting data.
                                                                                                                                            
  ### [cli-multiregnskab](https://github.com/holmconsulting/cli-multiregnskab)                                                              
  A command-line tool (`mr`) for interacting with the [Multiregnskab](https://multiregnskab.dk) accounting platform directly from the
  terminal. Built with TypeScript and Bun, it covers the full API surface: companies, customers, invoices, bank accounts, and products.     
  Pre-built binaries are available for macOS, Linux, and Windows.
